## Roadmap

Things to learn.

## Core Data Structures

- **Arrays & Strings**: Two pointers, sliding window, prefix sums, subarray problems.
- **Linked Lists**: Reversals, slow/fast pointers, detecting cycles.
- **Stacks & Queues**: Monotonic stacks, BFS using queues, sliding window maximum.
- **Hash Tables**: Frequency maps, caching, fast lookups.
- **Heaps / Priority Queues**: Top-k problems, merging sorted lists.
- **Trees**: Binary trees, BSTs, traversals (DFS, BFS), recursion, tree DP.
- **Graphs**: Representation (adjacency list/matrix), DFS, BFS, topological sort, shortest path algorithms (Dijkstra, Bellman-Ford).
- **Tries**: Prefix searches, autocomplete problems.
- **Disjoint Set / Union-Find**: Connected components, cycle detection.

## Core Algorithms & Patterns

- **Sorting & Searching**: Binary search, two-pointer search, custom sorting.
- **Dynamic Programming (DP)**:

  - 1D, 2D DP
  - Memoization vs bottom-up
  - State compression
  - Classic patterns: knapsack, LIS, matrix path, string edit distance

- **Backtracking / Recursion**: Subsets, permutations, combinations, N-Queens, factorials, Fibonacci.
- **Greedy Algorithms**: Interval scheduling, activity selection, coin change.
- **Graph Algorithms**: Dijkstra, BFS/DFS, Union-Find, Bellman-Ford, Floyd-Warshall.
- **Sliding Window / Two Pointers**: Subarray problems, substring problems.
- **Bit Manipulation**: XOR tricks, subsets using bits, mask DP.

## Additional Tools

- **Math / Number Theory**: Primes, GCD/LCM, modulo arithmetic, combinatorics.
- **String Parsing**: Regex, splitting, string manipulation.

## Notes

- LeetCode problems are a combination of **a pattern + a data structure + an optimization**
- Master the patterns and learn which data structure fits
